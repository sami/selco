2. Bugs Found

Bug #1 — Critical: Next Button Invisible

The bg-brand-blue Tailwind utility is used ~40 times across the codebase but never defined in the @theme block of global.css. The theme defines --color-primary: #003087 but no --color-brand-blue.

Impact: The wizard's Next button, Start Over button, progress bar, and unit toggle all have white text on a transparent background — effectively invisible to users.

Fix: Add --color-brand-blue: #003087; to the @theme block, or replace all brand-blue references with primary.

Bug #2 — Medium: Pluralisation Error

When a quantity equals 1, the display reads "1 bags", "1 bottles" instead of "1 bag", "1 bottle". Affects the Grout step, Primer step, and Summary table's Purchase qty column.

Fix: Add conditional pluralisation: n === 1 ? 'bag' : 'bags'
